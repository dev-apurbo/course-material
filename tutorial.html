How to Use the Course Portal

A quick guide to finding your Brac University course materials.

1 Searching for a Course

At the top right of the homepage, you will find a Search bar. Simply type the course code (e.g., "CSE422" or "MAT215") to filter the list instantly. The list will update as you type, showing only the courses that match your search.

2 Sorting the List

Next to the search bar, there is a Sort dropdown. You can use it to organize the courses:

  • Course Name (A-Z): Sorts courses in alphabetical order (default).
  • Course Name (Z-A): Sorts courses in reverse alphabetical order.

3 Accessing Main Course Materials

Each course is listed as a blue button. Clicking directly on the course name (e.g., CSE422) will open a new tab taking you to the main Google Drive folder containing the materials for that specific course.

4 Exploring Sub-Links

If a course has additional resources like playlists, practice sheets, or separate lab folders, you will see a + button on the right side of the course name.

  • Click the + button to expand and view the sub-links.
  • Click any of the gray sub-link buttons to access specific resources like YouTube playlists or Google Docs.
  • Click the - button to collapse the sub-links again.

5 Submitting New Materials

If you have course materials that are missing from the portal, or if you find broken links, you can contribute! Scroll to the bottom of the page and click on the email address in the footer: code.cse.bracu@gmail.com.

Clicking the email link will automatically open your device's default mail app (or Gmail web on PC) so you can send the files or links to the maintainers.
